### 1. Sign In
- Sign in to your n8n account.

### 2. Create a Project
- Go to Projects.
- Click Create Project.
- Enter a name and save.

### 3. Add a Workflow
- Open the project.
- Click Add Workflow.
- Name it Workflow 2.

### 4. Add First Step
- Click the “+” Add Node button in the workflow editor.

### 5. Search “Send Box Message”
- In the node search bar, type Send Box Message.
- Select and add the node to the canvas.

### 6. Search “AI Agent”
- Click “+” to add another node.
- Search for AI Agent.
- Add it to the workflow and connect it to the previous node.

### 7. Add Gemini Chat Model to AI Agent
- Open the AI Agent node settings.
- Under Model or Agent Model Settings:
- Search for Gemini Chat Model.
- Select the appropriate Gemini model (e.g., Gemini 1.5, Gemini Pro).
- Save the configuration.
