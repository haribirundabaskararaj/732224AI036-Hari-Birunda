### 1. Sign in
- Log in to your n8n account.

### 2. Create a Project
- Go to the Projects section.
- Click Create Project.
- Enter a name and save.

### 3. Add a Workflow
-Inside the project, click Add Workflow.
-Name your workflow (e.g., Workflow 1).

### 4. Add the First Step
- Open the workflow editor.
- Click the “+” (Add Node) button.

### 5. Search for Trigger
- In the search box, type Trigger.
- Select the desired trigger node (e.g., Webhook Trigger, Schedule Trigger, etc.).
- Add it to the canvas.

### 6. Add Email (Send Message)
- Click “+” again to add another node.
- Search for Email or Send Message.
- Select Email (Send) or the appropriate email-sending node.
- Add it to the workflow and connect it to the trigger.
