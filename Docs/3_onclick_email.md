### Create Workflow:
- Start a new workflow in your automation tool (e.g., n8n).

### Add Manual Trigger:
- Add a Manual Trigger node to start the workflow.

### Add Gmail Node: 
- Add a Gmail node and choose Send Email action.

### Configure Email: 
- Set To, Subject, and Body for the email.

### Connect Nodes:
 - Link the Manual Trigger node to the Gmail node.

### Execute: 
 - Click Execute Workflow, then trigger manually to send the email.
